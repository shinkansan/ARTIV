# how to use yolo_v4
> reference: https://github.com/AlexeyAB/darknet

### darknet 빌드

1. 다운
~~~(bash)
mkdir yolo_v4
cd yolo_v4
git clone https://github.com/AlexeyAB/darknet.git
~~~

2. 빌드
~~~(bash)
cd darknet/
mkdir build-release
cd build-release/
cmake ..
make
make install
~~~

### yolo_v4

~~~(bash)
./darknet detector demo ./cfg/coco.data ./cfg/yolov4.cfg ./yolov4.weights -c 0
~~~

yolov4.weights파일은 아래 링크에서 다운받을 수 있다.
