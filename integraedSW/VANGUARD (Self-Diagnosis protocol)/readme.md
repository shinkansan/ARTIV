  ### VANGUARD 개발 배경 및 설계   
  
  개발 과정 중 치명적인 브레이크 초깃값 설정 Issue를 발견하여 자가진단 프로토콜의 필요성을 느꼈다. 
  
  차량 시동을 걸기위해 브레이크를 사용자가 누르고 있는 상태에서 자율주행 모드를 키면, 
  차량 제어에 사용되는 브레이크 액츄에이터가 실제 브레이크와 떨어진 상태로 시작되는 Issue이다.
